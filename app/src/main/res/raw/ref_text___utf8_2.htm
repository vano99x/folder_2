Добавление менеджера.
Менеджер добавляется в базу Диамир, для него создаются контрольные точки на объектах, с которых будет осуществлено отслеживание.
Добавление сотрудника.
  1. У сотрудника должно быть заполнено поле NFC, номер метки, возможно узнать приложив метку в режиме отслеживания.
  2. Также у сотрудника должен быть указан клиент, на котором происходить отслеживание.
  3. После заполнения этих полей нужно синхронизовать, телефон.
Отслеживание.
Ввод логина менеджера, при успешном входе осуществляется синхронизация с базой.
После входа есть три доступных пункта меню “ Отслеживание”, “Синхронизация”, “Список чекинов”.
При клике на “ Отслеживание” появляется список доступных контрольных точек, после выбора точек открывается меню для выбора режима работы, после выбора режима работы – возможна фиксация событий с помощью меток NFC:
  1. “Начало работы” при фиксации события, отслеживается приход. В листы учёта записывается время прихода.
  2. “Окончание работы” в листы учёта записывается время ухода, и на основе разницы время прихода – время ухода, вычисляется отработанное время.
  3. ”Проверка”.
В случае отсутствия возможно зафиксировать событие с помощью поиска сотрудника.
Список чекинов.
Показывает список чекинов, в котором возможно задать фильтр по интервалу времени.
Синхронизация.
  1. Сотрудники из базы Диамир загружаются в Time Check.
  2. Чекины, зафиксированные при автономной работе, или те которые не удалось отправить, посылаются в базу.
